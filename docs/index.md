### Introduction  

# MPDCtrl

MPDCtrl is a client app for [MPD (Music player daemon)](http://www.musicpd.org/). 
   
- Requirement:  
MPDCtrl for desktop - 
Windows 10+ and a host running MPD.    

- Download:  
MPDCtrl for desktop -
[Download from Micorsoft Store](https://apps.microsoft.com/store/detail/mpdctrl/9NV2BBJ82BRX)
